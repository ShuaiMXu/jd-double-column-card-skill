## 二、内容家族 - 头像规则（按家族区分）

**v0.4 重要澄清**：头像不是所有卡都有，按家族区分：

| 家族 | 头像 | 说明 |
|---|---|---|
| 内容 | **必选** | UGC 心智锚点 |
| 点评 | **必选** | UGC + 决策证明 |
| 圈子 | **不展示** | 用圈子图标替代 |
| 商品 | **不展示** | 用店铺 logo 替代 |
| 新品 | **不展示** | 抹除主语（事件驱动）|

**头像规格**：16px 圆形（内容家族默认）

## 三、内容-图文卡结构（L2）

```
┌─────────────────────┐
│                     │
│      封面图         │  ← 3:4，不裁切，干净无叠字
│                     │
├─────────────────────┤
│ 标题文字（加粗）    │  ← 2 行以内，超出省略
├─────────────────────┤
│ 👤 头像 昵称  ♥ 赞数 │  ← 头像 16px 圆形
└─────────────────────┘
```

### 必含元素
- 封面图（3:4）
- 标题（加粗，1-2 行）
- 头像 + 昵称（左下）
- 互动数据（右下，图标 + 数字）

### 可选元素
- 互动模式切换（右下图标可以是 ❤️/👁/💬/⭐，见 L1-6）
- 标题中嵌 emoji 作钩子
- 封面叠字钩子（见第六节）

### 个人页特例
**业务线 = 个人中心**时，左上角**显式标图文 icon**（见 L1-1）。其他场景默认为空。

## 四、内容-视频卡结构（L2）

```
┌─────────────────────┐
│ ▶  [时长可选]        │  ← 左上角视频角标
│                     │
│   视频封面帧        │  ← 9:16 竖版
│                     │
├─────────────────────┤
│ 标题文字 2 行       │
├─────────────────────┤
│ 👤 头像 昵称         │
└─────────────────────┘
```

### 视频卡的三种亚型（按来源区分）

| 亚型 | 所属家族 | 封面来源 | 特征 |
|---|---|---|---|
| **内容视频卡** | 内容家族 | UGC 视频截帧 | 9:16 + 视频角标 + 头像昵称 |
| **商品视频卡** | 商品家族 | 静态产品图（非视频帧）| 3:4~1:1 + 视频角标 + 店铺+价格；**无头像** |
| **新品测评视频卡** | 新品家族 | 场景/测评画面 | 视频角标 + **必含时长** + 商品挂载浮条 |

**时长标识规律**（当前观察到的）：
- **测评视频** → 时长**必须**（用户判断"要花多少时间看"）
- **商品视频 / 内容视频** → 时长**可选**或省略

## 五、内容-直播卡结构（L2）

```
┌─────────────────────┐
│ 🔴 [观看数可选]      │  ← 左上角红色直播图标
│                     │
│   直播封面帧        │  ← 9:16 或 3:4 竖版
│   (可含动态画面)    │
│                     │
├─────────────────────┤
│ 标题文字            │
├─────────────────────┤
│ 👤 头像 昵称  ♥ 赞数 │
└─────────────────────┘
```

### 直播卡的三种亚型（按家族区分）

| 亚型 | 所属家族 | 关键识别 |
|---|---|---|
| **内容直播卡** | 内容家族 | 底部 = 头像+昵称+互动（UGC 底部不变量） |
| **商品直播卡** | 商品家族 | 右上角店铺胶囊 + 底部商品名/价格 |
| **新品直播卡入口** | 新品家族 | 底部事件胶囊（"重磅发布"）+ 互动激励 icon |

**关键辨识原则**：直播属性表现一致（左上角红图标），**家族身份由底部决定**（L1-5）。

### 内容直播卡的变体（在基础结构上叠加）

| 变体 | 新增元素 | 位置 |
|---|---|---|
| 红包雨变体 | 活动胶囊（"5000元 红包雨"）| 右上角 |
| 超头/达人变体 | 榜单/认证（"服饰美妆排行榜第1名"）| 标题下方（L1-3）|
| 预约变体 | 用户状态（"你预约的"）| 标题下方（灰色小字）|
| 沉浸式（达人/超头）变体 | 沉浸式场景摄影 + 活动叠字 + 左中个人化胶囊 | 封面整体重构 |

**变体可叠加组合**（超头 + 红包雨 + 预约可同时存在）。

## 六、封面文字叠加规范（分类展开）

内容家族的封面可以叠加文字/元素作为视觉钩子。按内容类型分类：

| 类型 | 典型示例 | 规范 |
|---|---|---|
| **钩子文案** | "超大生物""为什么它总挑事儿" | 字号 ≥ 20px，≤ 15 字，白字带阴影 |
| **弹幕气泡** | "星火·天冷，求保暖"（模拟用户弹幕）| 灰色/白色胶囊，位置灵活 |
| **趋势元素标签** | "麂皮面料""琥珀色针织面料" | 白色胶囊，体现关键属性 |
| **沉浸场景叠字** | "DIZERO BOSTON / 竞速美学" | 内容本身的设计的一部分 |
| **活动叠字** | "618 采销降价火热进行中" | 白字粗体，位置在封面下部 |
| **直播预告** | "4月8日 在蕉内直播间 等你" | 白字大号，直播卡常见 |

### 通用原则

- 字号 ≥ 20px（叠字才有视觉钩子作用）
- 字数 ≤ 15 字（钩子简洁有力）
- 位置灵活（图片下半部 / 中央 / 上部按内容决定）
- 白字必须带阴影或深色蒙版（避免浅色封面时失焦）
- **不能完全遮盖封面**（视觉权重要克制）
- **叠字不能代替卡下标题**（那就退化成"封面海报"）

## 七、常见结构差异模式

按 L1-5 家族 DNA 原则，以下为内容家族常见差异：

| 差异 | 说明 |
|---|---|
| 缺失头像 | 内容家族/点评家族必选；缺失即家族身份模糊 |
| 缺失作者昵称 | 同上 |
| 缺失互动数据 | 内容家族底部不变量被破坏 |
| 层级错乱 | 标题在封面上方 / 作者在标题上方 |
| 视频伪装 | 9:16 比例但无视频角标（业务上伪装成视频卡）|
| 直播伪装 | 红色氛围底但无"直"标签（仿直播氛围）|
| 身份混淆 | 内容卡底部出现商品信息压倒内容信号（像商品卡）|
